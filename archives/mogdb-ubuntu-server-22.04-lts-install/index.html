测试 ubuntu server 22.04 LTS 安装mogdb 5.0.11
使用的操作系统镜像是
https://releases.ubuntu.com/22.04/ubuntu-22.04.5-live-server-amd64.iso

装好操作系统后，把root登录打开了，方便后续操作。
